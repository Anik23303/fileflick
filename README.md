# ⚡ FileFlick - Universal File Converter

**100% Free · No Sign-up Required · Client-Side Privacy**

FileFlick is a modern web application and Chrome Extension that lets you convert images, documents, audio, and video instantly. Your files **never leave your device** for image conversions, making it the safest converter on the internet.

## ✨ Features
- 🖼️ **Image Conversion:** PNG, JPG, WebP, BMP, TIFF, AVIF, HEIC, and 40+ RAW formats.
- 📄 **Document Conversion:** PDF, DOCX, TXT, HTML, Markdown, ODT, RTF.
- 🎵 **Audio & Video:** MP3, WAV, AAC, MP4, AVI, MOV, MKV, and more.
- 🧩 **Chrome Extension:** Right-click any image on the web to convert it instantly.
- 🔒 **Zero Uploads:** All processing happens locally in your browser.

## 🚀 Quick Start
1. Clone the repo: `git clone https://github.com/your-username/fileflick.git`
2. Install dependencies: `npm install`
3. Run the dev server: `npm run dev`

## 🛠️ Tech Stack
- **Frontend:** React 18 + Vite + Tailwind CSS
- **Local Engine:** Canvas API + WebAssembly (FFmpeg.wasm)
- **Backend (Coming Soon):** Python FastAPI for heavy video/document processing
- **Extension:** Chrome Manifest V3

## 📦 Load the Chrome Extension
1. Open `chrome://extensions`
2. Enable **"Developer mode"**
3. Click **"Load unpacked"** and select the `/extension` folder.
